==========주의===========
사용 라이브러리가 32비트용입니다.
비쥬얼 스튜디오에서 빌드 시 32비트로 설정해주십시오.
========================


==========조작법==========
[ESC] 프로그램 종료
[1 / 3] 1인칭 / 3인칭 시점 변환
[WASD] 각 시점에 따른 이동
[R] 로봇과 플레이어 위치 초기화
[마우스] 1인칭 시점에서 좌우로 시야 회전
=========================


==========구현 내용=========
1. 바닥 구현
2. Player 캐릭터 구현
3. 로봇 네 개 구현
 - 주기적으로 방향을 바꿔서 이동한다.
 - 바닥의 경계에 충돌했을 때도 방향을 바꾼다.
4. 1 / 3인칭 시점 구현
 - 시점별로 wasd를 이용한 이동 방식이 다르다.
5. 로봇이 Player를 발견하면 방향을 바꿔 따라온다.
 - Player가 로봇에서 멀어지면 로봇은 다시 랜덤하게 움직인다.
6. 배경 이미지 삽입
==========================


===========보너스===========
1. 장애물(나무) 구현
 - 로봇과 충돌 체크한다.
 - 로봇은 나무와 만나면 방향을 바꾼다.
 - 로봇이 플레이어를 따라가는 중이라면 나무와 충돌해도 방향을 바꾸지 않고 그대로 통과합니다.
